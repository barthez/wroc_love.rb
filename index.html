Unicorn on Unix

@thorstenball

Complex HTTP server is based on 4 simple UNIX features:

  • fork() - create workers child process
  • signal() - send and handle signals from system
  • pipe() - send messages to workers (and self)
  • select() - wait on file for ready to read/write/error
